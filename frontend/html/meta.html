about the meta tag

The meta tag is used to provide metadata about the HTML document. Metadata is data (information) about data. The meta tag is always placed inside the head section of the HTML document.

Metadata will not be displayed on the page, but is machine parsable. Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata. Metadata can be used by browsers (how to display content or reload page), search engines (keywords), or other web services.

There are many types of meta tags, but the most common ones are:

  • Meta charset: This tag specifies the character encoding for the HTML document. Example:
    <meta charset="UTF-8">
  • Meta name="viewport": This tag is used to control the layout on mobile browsers. Example:
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Meta name="description": This tag provides a brief description of the page. Example:
    <meta name="description" content="This is a sample description of the webpage.">
  • Meta name="keywords": This tag specifies keywords relevant to the page. Example:
    <meta name="keywords" content="HTML, CSS, JavaScript, Web Development">
  • Meta name="author": This tag specifies the author of the document. Example:
    <meta name="author" content="John Doe">
  • Meta http-equiv="refresh": This tag is used to refresh the page or redirect to another URL after a specified time. Example:
    <meta http-equiv="refresh" content="30"> (refreshes the page every 30 seconds)
    <meta http-equiv="refresh" content="5;url=https://www.example.com"> (redirects to example.com after 5 seconds)
